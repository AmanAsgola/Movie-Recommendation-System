The Journey

0.005 → 0.2567
A 5,034% improvement.

20 tracked evaluation runs. Every decision logged. Every regression understood.

RUN #1 Precision@10: 0.0050

Broken Baseline — Random Noise

SVD returned global average rating for 99.9% of users — it never saw them during training. Evaluation was picking random users from 138K, but the model only knew 700. Almost identical to random guessing.

RUN #2 +1,400% ↑ 0.0750

Fix the Foundation — Correct Evaluation

Switched from scikit-surprise (Python 3.14 incompatible) to scipy SVD. Fixed evaluation to only test users the model actually knows. Added standard Precision@K denominator. Overnight +1,400%.

RUN #5 +144% ↑ 0.1833

Data Strategy — Train on Active Users

Stopped randomly sampling users. Selected top-12,000 by total rating count. More ratings per user = cleaner latent factors = better predictions. Single biggest improvement from data strategy alone.

RUN #10 +14% ↑ 0.2100

TMDB Metadata — Rich Content Signal

Fetched overview, cast, director, and keywords for all 27K movies from TMDB API. Two-stage retrieval: genre similarity first, then rich-text re-ranking. Content went from 20 generic genre tags to full plot descriptions.

RUN #16 +19% ↑ · GOOD ✅ 0.2500

Deep Learning — NCF/NeuMF + BPR Loss

Neural Collaborative Filtering (He et al. 2017 WWW Best Paper). GMF branch captures linear patterns; MLP captures non-linear taste. Trained with BPR pairwise ranking loss — directly optimises P@10. MPS-accelerated on Apple M5.

RUN #20 — CURRENT BEST +5,034% total ↑ 0.2567 GOOD ✅

Sentence-BERT + NCF + iALS Ensemble

Replaced TF-IDF with Sentence-BERT (384-dim semantic embeddings on MPS). Ensemble: 0.55×NCF + 0.45×iALS rank fusion. SBERT understands "psychological thriller ≈ dark crime drama" — TF-IDF cannot. Exceeds published NCF SOTA benchmark.

Technology

Four algorithms.
One perfect answer.

⚡

iALS

Implicit Alternating Least Squares. Treats liked movies (≥4.0 stars) as high-confidence positive signals. Confidence = 1 + 40 × rating. Pushes loved movies to the top of rankings.

RANKING OPTIMIZED
🧠

NCF/NeuMF

Neural Collaborative Filtering. GMF branch (linear) + MLP branch (non-linear). Trained with BPR pairwise ranking loss. Runs on Apple M5 MPS — 35% faster than CPU.

DEEP LEARNING · MPS
🔤

Sentence-BERT

all-MiniLM-L6-v2. 384-dim semantic embeddings for all 27K movies. Understands meaning — "psychological thriller ≈ dark crime drama". Cached to disk after first encoding.

SEMANTIC AI · TRANSFORMER
🔀

Ensemble Fusion

NCF + iALS rank fusion: 0.55×NCF + 0.45×iALS. Then hybrid: 0.70×collab + 0.30×SBERT content. Two models correct each other's errors — ensemble always beats either alone.
